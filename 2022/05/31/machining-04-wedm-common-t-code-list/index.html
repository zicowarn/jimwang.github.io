线切割数控加工T代码一览表

M代码 Sodick设备 备注
T80 开启电极丝驱动
T81 关闭电极丝驱动
T82 工作槽排水管关闭
T83 工作槽排水管开启
T84 开启高压喷水模式
T85 关闭高压喷水模式
T87 开启喷水泵
T88 关闭喷水泵
T89 转换至非沉浸式加工模式
T90 通过自动穿线机的顺序一进行断丝
T91 通过自动穿线机的顺序二进行断丝
T94 转换至沉浸式加工模式
T96 工作槽加注泵开启
T97 工作槽加注泵关闭
